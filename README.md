# End-To-End Weather Data Pipeline

An end-to-end data pipeline that extracts, transforms, and analyzes weather data using Python, SQL, and Excel. This project demonstrates data engineering concepts including API integration, data cleaning, feature engineering, and time-series analysis.

## Overview

This project builds a complete ETL (Extract, Transform, Load) pipeline using the Open-Meteo API. It retrieves both historical and forecast weather data, processes it into structured formats, and generates insights through analytical features and reporting.
